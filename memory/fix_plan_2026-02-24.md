# Fix plan — 2026-02-24

## Issues reported by user

1. **Brush ne marche pas** : cliquer/draguer sur la minimap n'a aucun effet visible (3D ne change pas, minimap reste statique).
2. **Sliders dans les nodes du Graph ne marchent pas** : impossible de bouger un slider à l'intérieur d'un node — le node se déplace au lieu.

## Root causes identifiées

### Bug 1 — Brush
Dans `TerrainStudio.jsx`, `applyBrush()` mute `genResult.data` **in place** (Float32Array). Le wrapper d'état change mais la **référence** de la `Float32Array` reste la même → les `useEffect` du viewer 3D et de la minimap qui dépendent de `data` ne re-firent pas.

### Bug 2 — Node sliders
Reactflow capture `mousedown`/`pointerdown` sur le node entier pour le drag. Les handlers `e.stopPropagation()` ne suffisent pas car reactflow écoute en capture phase. La bonne pratique reactflow est d'ajouter la classe CSS **`nodrag`** sur tout élément interactif (slider, select, button, checkbox).
